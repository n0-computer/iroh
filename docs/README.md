# Iroh Documentation Index

High-level and developer documentation for Iroh.

## Core Docs
- `../README.md` – Project overview, getting started.
- `relays.md` – General information about relays.
- `local_relays.md` – Conceptual overview of using local relays.
- `local-relay-dev.md` – Hands-on local relay + QUIC Address Discovery (QAD) setup, validation & troubleshooting.

## Auxiliary / Related Topics
- Relay development helper script: `../scripts/local-relay-setup.sh`
- Contribution guidelines: `../CONTRIBUTING.md`
- Code of Conduct: `../code_of_conduct.md`

## Suggested Order for New Contributors
1. Read the main `README.md`.
2. Skim `relays.md` for architecture.
3. Follow `local-relay-dev.md` to run a relay locally.
4. Use the helper script for quick re-setup cycles.
5. File issues with the reproducibility data described in `CONTRIBUTING.md`.

## Keeping This Index Updated
When adding a new doc to `docs/`, add a bullet here with a short description. This helps discoverability and avoids stale knowledge living only in PR descriptions.
