Expand description

Configuration for pushing metrics to a remote endpoint.

Fields§

§interval: u64

Push interval in seconds

§endpoint: String

Endpoint url where to push metrics

§service_name: String

Name of the service you’re exporting metrics for generally metrics_exporter is enough for use outside of production deployments

§instance_name: String

The name of the instance you’re exporting metrics for ie. username-laptop, username-phone, etc. Should be reasonably unique

§username: Option<String>

Username for basic auth

§password: String

Password for basic auth
