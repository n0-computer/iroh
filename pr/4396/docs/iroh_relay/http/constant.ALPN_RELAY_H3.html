ALPN protocol identifier for H3 relay connections.

Uses the standard HTTP/3 ALPN. The relay connection is established with a WebTransport CONNECT request (via web-transport-proto); the relay protocol version is carried in that request’s subprotocol field, mirroring the Sec-WebSocket-Protocol header used on the HTTP/1.1 WebSocket path.
