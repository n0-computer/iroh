Crate iroh_dns_server 

Source
Expand description

A DNS server and pkarr relay.

Server combines a DNS server (UDP and TCP) with an HTTP/HTTPS server into a single process. Clients publish self-signed DNS records as pkarr signed packets at PUT /pkarr; the server persists them and answers DNS queries for the published names, including DNS-over-HTTPS at /dns-query.

With the mainline fallback enabled, keys missing from the local store are looked up on the BitTorrent mainline DHT.

§Example

use iroh_dns_server::{Server, config::Config};
let config = Config::load("config.toml").await?;
let server = Server::bind(config).await?;
server.join().await?;
