Expand description

Trait for structs that can be converted into EndpointIdResolutionSystems.

This trait is implemented on builders for ERS’s. Any type that implements this trait can be added as a ERS in Builder::ers.

Any type that implements EndpointIdResolutionSystem also implements IntoErs.

Iroh uses this trait to allow configuring the set of ERS’s on the endpoint builder, while providing the EIR services access to information about the endpoint to IntoErs::into_ers.
